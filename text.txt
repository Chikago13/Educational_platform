# Educational_platform

Образовательная платформа

    1. Система тестирования
        a. Возможность создавать/редактировать/удалять тесты пользователям с определенными правами
        b. Возможность “делиться” тестами между преподавателями
        c. Тесты разных видов(с выборами вариантов ответа, вписывание текста в поле и т.д.)
        d. Разделение на курсы и темы внутри курсов
        e. Отслеживание прогресса по темам/курсам (профиль студента)
        f. Возможность блокировки тестов в зависимости от пройденных тестов или пройденных тем.
        g. Статьи с материалами для ознакомления
        h. Рекомендации курсов для студента
	
    2. Юзерс
        a. Объединение студентов в группы; добавление и удаление групп к преподавателям
        b. Отслеживание прогресса студентов со стороны преподавателя (возможность просматривать профиль студентов)
        c. Профиль преподавателя
        d. Чат с преподавателем*
        e. Аутентификация на сайте с отправкой письма на почту*

    3. Нефункциональные требования
        a. Язык и фреймворки: Python, Django 4.x, DRF
        b. Docker для контейнеризации. 
        c. JWT для Auth
        d. Виртуальное окружение: poetry
        e. Linters  (black, isort, flake8)
        f. GIT
        g. БД удаленно PostgreSQL
        h. * redis, celery для очередей
